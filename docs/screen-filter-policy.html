1. 我们收集的信息

屏幕滤镜是一款离线应用，使用过程中会监测光线传感器、屏幕点亮状态、以及正在使用的应用，但这些工作都会在您的手机上完成，任何数据都不会被发送到服务器。这也就意味着，我们也无法提供数据同步服务。

 

2. 权限使用

我们将在运行应用程序使用以下权限：

l android.permission.SYSTEM_ALERT_WINDOW 

l 显示滤镜叠加层（适用于Android 8.0以前的系统版本）。

l android.permission.SYSTEM_OVERLAY_WINDOW

l 显示滤镜叠加层(适用于Android 8.0 及以后的系统版本)。

l android.permission.WRITE_SETTINGS

l 改变系统设置的屏幕亮度。

l android.permission.WRITE_SECURE_SETTINGS

l 改变系统设置的屏幕亮度。

 

我们不会对您的系统设置做出不必要的修改，并且您在使用本应用过程中造成的任何更改，都可以在卸载本应用后还原。

 

 

3. 数据存储

您在使用应用过程中产生的个性化数据，我们都会在您的手机上以安全的方式存储，确保不会被其它应用程序利用，这些数据会在卸载此应用时被删除，不会被发送到我们的服务器，也不会被分享给第三方机构。

 

 

4. 您的权限

如果您对我们的应用权限和安全策略有任何顾虑和疑惑，可随时通过邮箱向我们反馈以及获得帮助。

5. 变更

我们可能适时修订本《隐私政策》的条款。当变更发生时，我们会在版本更新时向您提示新的《隐私政策》，并向您说明生效日期。请您仔细阅读变更后的《隐私政策》内容，若您继续使用我们的服务，即表示您同意我们按照更新后的《隐私政策》处理您的个人信息。

6. 未成年人保护

我们鼓励父母或监护人指导未满十八岁的未成年人使用我们的服务。我们建议未成年人鼓励他们的父母或监护人阅读本《隐私政策》，并建议未成年人在提交的个人信息之前寻求父母或监护人的同意和指导。
